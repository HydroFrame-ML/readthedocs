hydrodata.data_catalog.data_access.get_file_paths(*args, **kwargs) → List[str]

Get a list of file paths from the data catalog identified by the filter options.

Parameters:

  • args – Supports one optional positional parameters that must be a dict with filter options.

  • kwargs – Supports multiple named parameters with filter option values.

Returns:

A list of absolute path names to files in the data catalog entry after looping through time range and replacing substitution keys.

Raises:

  • ValueError if no data catalog entry is found matching the options provided. –

  • ValueError if two different data catalog entries match the options provided. –

A filter option either as a named argument or an entry in the dict passed as the first argument may be any metadata attibute in a data catalog entry or any attribute below.

  • time: A time in the form YYYY-MM-DD. Used to substitute time keys in the file path.

  • start_time: A time in the form YYYY-MM-DD. Start of date range for time keys in the file path.

  • end_time: A time in the form YYYY-MM-DD. End of date range for time keys in the file path.

  • site_id: An observation point site id.

If both time and start_time is specified then start_time is used.

For example, the following are equivalent:
options={

“dataset”: “baseline”, “grid”:”conus1”, “start_time”:”2023-01-15”, “variable”:”water_table_depth”

}

paths = get_file_paths(options)

paths = get_file_paths(dataset=”baseline”, grid=”conus1”, start_time=”2023-01-15”, variable=”water_table_depth”)

paths = get_file_paths(**options)

hydrodata.data_catalog.data_access.get_metadata(*args, **kwargs) → dict

Get the metadata associated with the specified filter options.

Parameters:

  • args – Supports one optional positional parameters that must be a dict with filter options.

  • kwargs – Supports multiple named parameters with filter option values.

Returns:

A dict with all the metadata of the data catalog entry. This also includes columns data_catalog_entry and datapath.

Raises:

  • ValueError if no data catalog entry is found matching the options provided. –

  • ValueError if two different data catalog entries match the options provided. –

An option value may be any attribute in the data catalog entry metadata.
