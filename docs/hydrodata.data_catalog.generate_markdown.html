hydrodata.data_catalog.generate_markdown module

Functions to generate markdown files(s) for tables in the hydrodata_catalog.yaml file.

hydrodata.data_catalog.generate_markdown.generate_markdown_tables(output_dir='.')

Generate markdown files to document the contents the data catalog. :param output_dir: The directory to generate the markdown files (default is current dir).

Raises:

ValueError if the table does not exist in the data_catalog. –

Generates a markdown file in the output directory with the name {table}.md for each generated table. The generated file is a markdown file containing a table with the rows and columns of the table information from the data_catalog.
